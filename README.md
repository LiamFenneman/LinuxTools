# LinuxTools
### By Liam Fenneman

# Tool 1 - git.sh

### Usage

`git.sh "<message>"`

### Description

This tool is used to automatically add all files in a given folder to the git staging area.
Then commits using the given message. And finally pushes the commit to the 'main' branch.

# Tool 2 - gccr.sh

### Usage

`gccr.sh <file_name>`

*NOTE: file_name doesn't include an extension.*

### Description

This tool will compile a given .c file into an executable and then run it.
